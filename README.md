# Cognifyz Machine Learning Internship

This repository contains the projects completed during my Machine Learning Internship at Cognifyz Technologies.

## Projects

### Task 1: Restaurant Rating Prediction

**Objective:** Predict restaurant ratings using machine learning.

**Tech Stack:**

* Python
* Pandas
* Scikit-Learn
* Matplotlib

**Model Used:**

* Decision Tree Regressor

**Results:**

* Mean Squared Error (MSE): 0.0564
* R² Score: 0.9752

---

### Task 2: Restaurant Recommendation System

**Objective:** Build a content-based recommendation system based on cuisine preferences and price range.

**Techniques Used:**

* TF-IDF Vectorization
* Cosine Similarity

**Features:**

* Personalized restaurant recommendations
* Cuisine-based filtering
* Price-range matching

---

### Task 3: Cuisine Classification

**Objective:** Classify restaurants into cuisine categories using machine learning.

**Model Used:**

* Random Forest Classifier

**Results:**

* Accuracy: 31.68%

**Evaluation Metrics:**

* Accuracy
* Precision
* Recall
* F1-Score

---

### Task 4: Location-Based Analysis

**Objective:** Analyze restaurant distribution across different cities and locations.

**Analysis Performed:**

* Restaurant concentration by city
* Average ratings by city
* Price range analysis
* Cuisine popularity analysis
* Data visualization
